@par Description

This directory contains binary files (*.hex) of the "STemWin Firmware Package" examples. 

You can use any in-system programming tool to reprogram the example using the 
provided binary files, as described below:

   - Connect the STM32xx-EVAL board to a PC using ST-Link/V2 debugger or any other compatible HW
   - Use "*.hex“ file with STM32 ST-LINK Utility to program the application firmware.        
     Note: you can download STM32 ST-LINK Utility from http://www.st.com
